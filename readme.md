Function Visualization

What the program does:
    This program visualizes 5 different mathematical functions using Python.
    The program executes the chosen function and then plots the graphs.

How to run the program:
    The first step is to install the {numpy} and the {matplotlib} libraries using the "pip install" command.
    After that the program can be ran normally with "python main.py".

Functions that are plotted:
    The program plots the following functions:

    Linear function : f(x) = ax + b
    Quadratic function : f(x) = ax² + bx + c
    Sine function : f(x) = sin(x)
    Exponential function : f(x) = eˣ
    Cubic function : f(x) = x³
